# Sweater Landing Page

A modern and visually appealing landing page for a sweater collection built using **HTML5** and **CSS3**. The project focuses on creating a clean, minimal, and fashion-inspired user interface with an elegant layout.

## ✨ Features

* Clean and modern landing page design
* Responsive viewport setup
* Minimal navigation bar
* Large hero section with product images
* Product details section
* "Shop the Look" call-to-action
* Font Awesome icons integration
* Simple and easy-to-understand code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome
